## How to sign

You accept this Agreement in either of the following ways:

1. **Per-commit sign-off (DCO-style).** Add a `Signed-off-by` line to every
   commit, certifying the Developer Certificate of Origin and your acceptance of
   this CLA:

   ```

   ```

   Create it automatically with `git commit -s`.

2. **One-time CLA acknowledgement.** State the following in your first pull
   request (or via an automated CLA-assistant check, if configured):

   > submit my Contributions under its terms.
